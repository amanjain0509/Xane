# README #

Xane API.

### What is this repository for? ###

This repository contains the API code for XaneAI.

### How do I get set up? ###

```
git clone https://bitbucket.org/xaneai/xane-api
cd xane-api
npm install
cd node_modules
ln -s ../app app
cd app
mkdir logs
touch logs/morgan.log
cd ../..
node server.js
```
